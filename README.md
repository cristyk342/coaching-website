# Life Coaching Website

A beautiful, interactive life coaching website featuring a skill tree carousel visualization that shows client transformation journeys.

## Features

- Brené Brown-inspired design aesthetic
- Interactive skill tree carousel showing client progression paths
- Horizontal flow from problem to goal
- Warm earth tone color palette (#d4a373 copper, #f9f6f1 cream)
- Responsive design
- Georgia serif typography throughout

## Design Credits

Design inspired by Brené Brown's website aesthetic with custom interactive skill tree visualization.

## How to Use

Simply open `index.html` in a web browser to view the website.

## Deployment

This website can be deployed to:
- GitHub Pages
- Netlify
- Vercel
- Any static site hosting service
